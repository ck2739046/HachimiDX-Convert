================================================================================
                        HachimiDX - 安装指南
================================================================================


第一步: 安装依赖（仅需操作一次）

1. 打开 install 文件夹
2. 双击运行 install.bat
3. 等待安装完成
4. 安装完成后关闭窗口即可

注意：安装只需执行一次，以后无需再次运行


--------------------------------------------------------------------------------

第二步: 启动软件


1. 返回项目根目录
2. 双击 HachimiDX.exe 启动软件


--------------------------------------------------------------------------------

第三步: 模型配置（仅需操作一次）


首次使用需要配置模型推理后端：

1. 进入「设置」页面
2. 在「模型推理」分区中，找到「模型后端」选项
3. 选择合适的后端：

   - CPU:      最慢，但无需 GPU
   - TensorRT: 最快，需要 NVIDIA 显卡
   - DirectML: 速度中等，需要兼容的显卡（NVIDIA/AMD/Intel）
   
4. 点击「检测」按钮检查环境
5. 点击「转换模型」按钮
6. 等待转换完成 (此过程可能需要 10 分钟)

注意：此步骤只需执行一次，模型转换成功后即可正常使用软件。


--------------------------------------------------------------------------------

完成！
