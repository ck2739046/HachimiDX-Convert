================================================================================
                        HachimiDX - 安装指南
================================================================================


第一步: 安装依赖（仅需操作一次）

1. 打开 install 文件夹
2. 双击运行 install.bat
3. 按照屏幕提示完成安装
4. 安装完成后关闭窗口即可

注意：安装只需执行一次，以后无需再次运行


--------------------------------------------------------------------------------

第二步: 启动软件


1. 返回项目根目录
2. 双击 HachimiDX.exe 启动软件


--------------------------------------------------------------------------------

第三步: 设置语言（仅需操作一次）


首次使用建议先设置界面语言：

1. 进入「设置」页面
2. 在「通用设置」分区中，找到「语言」选项
3. 选择你偏好的语言（简体中文 / English）
4. 点击页面最底部的「保存」按钮
5. 重启 HachimiDX 使语言设置生效

注意：保存后需要重启应用才能看到界面语言的变化。


--------------------------------------------------------------------------------

第四步: 模型配置（仅需操作一次）


首次使用需要配置模型推理后端：

1. 进入「设置」页面
2. 在「模型推理」分区中，找到「模型后端」选项
3. 选择合适的后端：

   - CPU:      最慢，但无需 GPU
   - TensorRT: 最快，需要 NVIDIA 显卡
   - DirectML: 速度中等，需要兼容的显卡（NVIDIA/AMD/Intel）
   
4. 点击「检测」按钮检查环境
5. 点击「转换模型」按钮
6. 等待转换完成 (此过程可能需要 10 分钟)

注意：此步骤只需执行一次，模型转换成功后即可正常使用软件。


--------------------------------------------------------------------------------

第五步: ffmpeg 编解码配置 (仅需操作一次)


首次使用需要配置 ffmpeg 编解码器：

1. 进入「设置」页面
2. 在「ffmpeg硬件加速」分区中，找到「编码」选项，选择合适的选项
3. 如果你不太了解，点击「自动检测最佳配置」按钮
4. 记得点击页面最底下的「保存」按钮

注意：此步骤只需执行一次


--------------------------------------------------------------------------------

完成！
